Max Payne 3 Script Hook


About

Max Payne 3 Script Hook allows you to use script mods (.asi) on the PC.

SDK

The SDK allows developers to make their own scripts. A simple example is included.

Donate

Want to donate? Click the button below!

I strongly think all mods should be free but every donation would help a lot and is greatly appreciated.

Requirements

  • Visual C++ Redistributable 2017 (x86)

    • Supported versions

  • 1.0.0.114 (non-Steam)
  • 1.0.0.216 (Steam)
  • other versions should also work but they're untested

    • Download & Support

    The script hook can be downloaded here (current version: 1.0.17.0).

    The trainer can be downloaded here.

    To download scripts or if you need help, join the official Max Payne 3 Modding Discord.

    Installation

    Copy ScriptHook.dll and dinput8.dll to the game's main folder (where MaxPayne3.exe is located)
